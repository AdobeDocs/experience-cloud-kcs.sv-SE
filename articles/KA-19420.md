# Så här kontrollerar du databaslagring

## Beskrivning

Så här kontrollerar du databaslagring för ACC-/ACS-miljöer

## Upplösning


Du kan använda Kontrollpanelen för att kontrollera hur länge databasen används jämfört med att lagra kostnadsfritt dygnet runt. Se [https://experienceleague.adobe.com/docs/control-panel/using/performance-monitoring/database-monitoring/database-monitoring.html?lang=en](https://experienceleague.adobe.com/docs/control-panel/using/performance-monitoring/database-monitoring/database-monitoring.html?lang=en) om du vill veta mer.



Obs! Du måste ha administratörsbehörighet i Experience Cloud för att kunna komma åt Kontrollpanelen.
