# API-importfelrelation 0 för klassificeringar är inte inställd för import av data

## Beskrivning


Vad är orsaken till API-importfelet&quot;Relation 0 har inte konfigurerats för import av data&quot; i API:t för klassificeringar (SAINT)?


## Upplösning


Felmeddelandet&quot;Relation 0 är för närvarande inte inställd på att importera data&quot; visas när variabeln &quot;product&quot; används med det övre&quot;Product&quot;. Klassificerings-API accepterar bara variabeln med &quot;p&quot; i det gemena fallet som &quot;product&quot;.
