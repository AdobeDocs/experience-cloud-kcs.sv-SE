# Distributionen av pipeline misslyckas under byggsteget

## Beskrivning {#description}

## Upplösning {#resolution}


- leta upp filen pom.xml i roten för Git och hitta till exempel nodversionen: nodeVersionv10.13.0/nodeVersion

- leta upp ui.frontend/package.json och lägg till i ett devDependencies-objekt som redan finns: till exempel:


```
"devDependencies": {"@types/node": "10.13.0",...}
```


Obs!
1. v från nodeVersion10.13.0 får inte kopieras.
2. Nodversionen kan vara en annan i ditt fall. Till exempel: 12.22.7
