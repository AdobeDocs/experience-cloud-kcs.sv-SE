## Upplösning {#resolution}


Det här felet är ett kundkodsfel som måste åtgärdas genom att nodversionen uppdateras i en misslyckad modul med hjälp av npm. För att kunna göra den här ändringen måste kunden göra följande:

1. Hitta den nodversion som används i reaktorn pom.xml. Nedanför exempel = 16.14.0 `pom.xml`:<br>

   ```
   <configuration>
           <nodeVersion>v16.14.0</nodeVersion>. 
           <npmVersion>8.3.1</npmVersion>
       <configuration>
   ```


2. Lägg till den här nodversionen som <b>devDependencies</b> i `package.json` filen för modulen som inte gick att skapa. I exemplet ovan är det UI-teckensnitt.<br>


   FÖRE - `package.json`:


   ```
   "devDependencies": {
       "@babel/core": "7.0.0",
       "@babel/plugin-proposal-class-properties": "7.3.3",
       "@babel/plugin-proposal-object-rest-spread": "7.3.2",
       "@material/tab-bar": "12.0.0",
       },
   ```




   EFTER - `package.json`:


   ```
   "devDependencies": {
       "@types/node": "16.14.0",                     <b></b>*THIS LINE WAS ADDED
       "@babel/core": "7.0.0",
       "@babel/plugin-proposal-class-properties": "7.3.3",
       "@babel/plugin-proposal-object-rest-spread": "7.3.2",
       "@material/tab-bar": "12.0.0",
       },
   ```
