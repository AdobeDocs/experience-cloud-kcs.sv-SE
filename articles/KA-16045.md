# Behöver hjälp med SDK 5.0-implementering

## Beskrivning


Efter SDK5-integrering når inte registreringsanropet ACS-instansen.


## Upplösning


Funktionen&quot;Campaign.registerExtension();&quot; implementerades inte i IOS App, vilket ledde till det här problemet.

För referens, se: [https://aep-sdks.gitbook.io/docs/using-mobile-extensions/adobe-campaign-standard](https://aep-sdks.gitbook.io/docs/using-mobile-extensions/adobe-campaign-standard)
